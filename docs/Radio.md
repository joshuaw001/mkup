# The `<radio>` tag

***

## usage

Creates a *single* radio button

***

## syntax

```html

<gui>
    <radio attributes...>
        True
    </radio>
</gui>

```

***

## attributes

The `<radio>` tag has 2 attributes:

1) data-color
2) data-img
3) data-btn1-color\* (inner circle)
4) data-btn2-color\* (outer circle)
