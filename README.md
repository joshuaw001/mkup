# MKUP: the upgraded markup language for html :)

***

## Features:

### amazing GUI tags:

1) `<slider>`
2) `<diagram>`
3) `<window>`
4) `<board>`
5) `<gui>`

All of these tags have ending tags

***

### Easy to learn:

`html
<gui>
  <diagram data-img=“Diagram.png”></diagram>
  <slider data-start=“1” data-end=“10”> 5 </slider>
</gui>
`
